# Quickstart: Agent Graph UI

## Prerequisites
- Java 21
- Node.js (LTS)

## Run backend
1. Start the Spring Boot app for `multi_agent_ide`.
2. Confirm the text/event-stream endpoint is reachable.

## Run frontend
1. From `multi_agent_ide/fe`, install dependencies.
2. Start the dev server.
3. Open the UI in a browser.

## Verify
- Trigger a graph execution and confirm nodes render and update.
- Pause or interrupt a node and confirm the UI reflects the action.
